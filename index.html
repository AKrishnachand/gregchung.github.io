Computation-Ready Experimental Metal-Organic Frameworks

Enabling accelerated discovery of high-performing metal-organic frameworks

View the Project on GitHub gregchung/CoRE-MOFs

All bug reports regarding the dataset should be sent to: greg.chung@pusan.ac.kr

Download the database of crystal structures by clicking the DROPBOX links below:

  • CORE-MOF-DB-09152014 (submission through Feb. 2014)
  • CoRE-MOF-DB-08182015 (>2,900 structures with DDEC charges)

You can access ~4,700 MOF structres by downloading the CORE-MOF-DB-09152014.

If you want to access entire crystal structures (5,109), you need to send an e-mail to: greg.chung (at) pusan.ac.kr

We'd appreciate if you could cite the following papers if you find the dataset useful:

Computation-Ready, Experimental Metal–Organic Frameworks: A Tool To Enable High-Throughput Screening of Nanoporous Crystals Yongchul G. Chung, Jeffrey Camp, Maciej Haranczyk, Benjamin J. Sikora, Wojciech Bury, Vaiva Krungleviciute, Taner Yildirim, Omar K. Farha, David S. Sholl, and Randall Q. Snurr, Chemistry of Materials 2014 26, 6185-6192 ( for the original database contains 5,109 structures)

A Comprehensive Set of High-Quality Point Charges for Simulations of Metal–Organic Frameworks Dalar Nazarian, Jeff S. Camp, David S. Sholl, Chemistry of Materials 2016 28, 785-793 (for DDEC charges)
