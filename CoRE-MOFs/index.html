Computation-Ready, Experimental Metal-Organic Frameworks Database

High-throughput computational screening of metal-organic frameworks rely on the availability of disorder-free atomic coordinate files which can be used as input to simulation software packages. Our group currently maintains the CoRE MOF database and continues to collect, curate, and analyze this unique class of material.
